## 8.1 기술적 준비

### 시스템 구성 확인

- [ ] **모든 토픽 및 작업 테스트 완료**
  - 토픽 트리거 확인
  - 액션 정상 실행 검증
  - 대화 흐름 자연스러움
  - 테스트 커버리지: 최소 80% 이상

- [ ] **인증 및 권한 설정 검증**
  - Entra ID / OAuth 설정 완료
  - API 권한 승인 완료
  - 테스트 사용자 로그인 확인

- [ ] **외부 시스템 연동 테스트**
  - SharePoint 문서 검색 정상
  - Dataverse 데이터 조회 정상
  - Power Automate Flow 실행 정상

- [ ] **성능 테스트 통과**
  - P95 응답 시간 < 5초
  - 동시 사용자 100명 처리
  - 오류율 < 1%

- [ ] **보안 취약점 점검 완료**
  - 인증 우회 불가능
  - SQL Injection 방어
  - XSS 공격 차단

- [ ] **백업 및 복구 계획 수립**
  - Copilot 구성 export
  - 복구 절차 문서화
  - 복구 테스트 1회 이상 수행

- [ ] **모니터링 및 알림 설정**
  - Application Insights 연결
  - 알림 규칙 설정
  - 대시보드 구성

---

## 8.2 비즈니스 준비

- [ ] **사용자 교육 자료 준비**
  - 사용자 가이드 (PDF)
  - 빠른 시작 가이드
  - 비디오 튜토리얼

- [ ] **헬프데스크 지원 프로세스 수립**
  - 지원 티켓 시스템
  - 헬프데스크 교육
  - 트러블슈팅 가이드

- [ ] **에스컬레이션 프로세스 정의**
  - Level 1: Copilot 자동 응답
  - Level 2: 일반 지원팀
  - Level 3: 전문가 팀
  - Level 4: 개발팀

- [ ] **비상 연락망 구성**
  - 온콜 엔지니어 지정
  - 백업 담당자 지정
  - 커뮤니케이션 채널 설정

- [ ] **SLA 정의**
  - 가용성: 99.5% uptime
  - P95 응답 시간 < 5초
  - Critical 이슈: 1시간 내 대응

---

## 8.3 운영 준비

- [ ] **운영 매뉴얼 작성**
  - 시스템 아키텍처
  - 일상 운영 절차
  - 문제 해결 가이드

- [ ] **장애 대응 절차 수립**
  - 인시던트 대응 프로세스
  - 에스컬레이션 경로
  - 복구 절차

- [ ] **정기 점검 일정**
  - 일일: 가용성 확인
  - 주간: 성능 추이 분석
  - 월간: 용량 계획 리뷰
