## 개요

Copilot Studio 에이전트 배포의 성공을 위해서는 명확한 목표 설정이 필수입니다. 이 문서는 비즈니스 목표, 측정 가능한 KPI, 그리고 명확한 범위 설정 방법을 안내합니다.

---

## 1.1 비즈니스 목표

에이전트가 해결해야 할 구체적인 비즈니스 문제를 명확히 정의합니다.

### 구체적 예시

- 고객 문의 응답 시간을 50% 단축
- IT 헬프데스크 티켓 30% 감소
- 신입 사원 온보딩 프로세스 자동화
- 내부 문서 검색 시간 70% 절감

### 목표 정의 템플릿

```
문제: [현재 해결해야 할 문제]
목표: [달성하고자 하는 결과]
대상: [서비스 대상 사용자]
기대효과: [예상되는 비즈니스 임팩트]
```

**예시:**
```
문제: 직원들이 HR 정책을 찾는데 평균 15분 소요
목표: HR 정책 검색 시간을 3분 이내로 단축
대상: 전체 직원 (500명)
기대효과: 월 125시간 생산성 향상 (500명 × 10분 절감 × 월 1.5회)
```

## 1.2 성공 지표 (KPI)

측정 가능한 지표를 설정하고 정기적으로 추적합니다.

### 정량적 지표

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| **응답 정확도** | 90% 이상 | 올바른 답변 비율, 단계별 정확도 수치 조정 필요; 1단계 60%, 2단계 70% ... |
| **응답 시간** | 3초 이내 | P50 응답 시간 |
| **완결률** | 80% 이상 | 에이전트가 해결한 대화 비율 |
| **사용자 만족도** | 4.5/5 이상 | 별점 또는 피드백 점수 |
| **일일 활성 사용자** | - | DAU 추적 |
| **세션당 대화 수** | - | 사용자 참여도 측정 |

### 정성적 지표

- 사용자 피드백 분석
- 에스컬레이션된 케이스의 유형 및 빈도
- 에이전트 개선 제안 사항

### 측정 방법

**Application Insights 쿼리 예시:**

```kql
// 평균 응답 시간
customEvents
| where name == "ConversationEnd"
| extend Duration = todouble(customDimensions["duration"])
| summarize AvgDuration = avg(Duration), P95Duration = percentile(Duration, 95)
```

```kql
// 사용자 만족도
customEvents
| where name == "UserFeedback"
| extend Rating = toint(customDimensions["rating"])
| summarize 
    AvgRating = avg(Rating),
    PositiveRate = countif(Rating >= 4) * 100.0 / count()
```

## 1.3 범위 설정

에이전트가 담당할 업무 범위와 제한 사항을 명확히 구분합니다.

### 범위 내 (In-Scope)

✅ **포함되는 기능:**
- 자주 묻는 질문 (FAQ) 응답
- 문서 검색 및 정보 제공
- 간단한 데이터 조회 및 확인
- 프로세스 안내 및 가이드 제공

### 범위 외 (Out-of-Scope)

❌ **제외되는 기능:**
- 복잡한 의사결정이 필요한 경우
- 민감한 개인정보 처리
- 승인이 필요한 업무
- 법적 자문 또는 전문가 판단이 필요한 경우

### 명확한 한계 설정

**토픽 설정 예시:**

```yaml
# 토픽에서 범위 외 처리 예시
additionalInstructions: |
  너는 다음 범위 내에서만 답변할 수 있다:
  - HR 정책 및 복리후생 안내
  - 휴가 신청 절차 안내
  
  다음 질문은 답변하지 않고 HR 담당자에게 연결한다:
  - 급여 관련 문의
  - 개인 민감 정보 변경
  - 징계 또는 법적 문제
```

### 범위 문서화 템플릿

| 카테고리 | 범위 내 | 범위 외 | 에스컬레이션 경로 |
|---------|---------|---------|------------------|
| HR 정책 | 정책 조회, 일반 안내 | 급여, 인사 평가 | HR 담당자 |
| IT 지원 | 비밀번호 재설정, 일반 문의 | 하드웨어 교체, 시스템 장애 | IT 헬프데스크 |
| 시설 관리 | 회의실 예약, 주차 안내 | 시설 수리, 보안 문제 | 시설팀 |
