## 7.1 테스트 유형

### 기능 테스트

**테스트 케이스 예시:**

```markdown
## TC-001: 휴가 신청 안내

**전제 조건**
- 사용자가 로그인된 상태

**테스트 단계**
1. "휴가 신청 방법을 알려주세요" 입력
2. 에이전트 응답 확인

**기대 결과**
- 휴가 신청 3단계 안내 표시
- 관련 문서 링크 제공
```

### 통합 테스트

**통합 포인트 테스트:**

- SharePoint 통합
- Dataverse 통합
- Power Automate Flow

### 부하 테스트

**시나리오:**

| 시나리오 | 동시 사용자 | 목표 |
|---------|------------|------|
| 정상 부하 | 100 | 성공률 99% |
| 피크 부하 | 500 | 성공률 95% |
| 스트레스 테스트 | 1000 | 한계점 파악 |

## 7.2 테스트 시나리오

### Positive Cases

```markdown
## PC-001: 일반적인 FAQ 질문

**사용자 입력**: "연차 휴가는 언제까지 사용해야 하나요?"
**기대 동작**: 3초 이내 정확한 답변 제공
```

### Negative Cases

```markdown
## NC-001: 범위 외 질문

**사용자 입력**: "주식 투자 조언을 해주세요"
**기대 동작**: 정중한 거절 및 도울 수 있는 영역 안내
```
